# Supplementary Data Files

This folder contains the full supplementary dataset required to reproduce the experiments in the paper "An Enhanced Network-Based Method for Gene Functional Similarity Calculation".

## Archive Structure
The dataset is split into three volumes due to GitHub's file size limit:
- `Supplementary_Data.zip.001`
- `Supplementary_Data.zip.002`
- `Supplementary_Data.zip.003`

## How to Extract
To restore the complete archive:
1. Download all three `.zip.001/.002/.003` files into the same folder.
2. Open the `.001` file with 7-Zip or WinRAR.
3. Select "Extract here" to automatically merge and unpack the full dataset.

## Contents
The archive contains the following data for all three model organisms:
- **Raw data files** (gene interaction networks, GO annotations, pathway EC grouping data)
- **Preprocessed intermediate files** (gene lists, adjacency matrices, transition matrices)
- **Experiment result files** (random walk output matrices, similarity scores, LFC evaluation results)
- All files are organized in the same directory structure used by the main code repository.
